2026年1月24日：谁在动我的鼠标？

我今天坐在电脑前，发现光标自己在慢慢移动。它停在我的照片上，左右晃动，就像有人在隔着屏幕抚摸我的脸。

尤舍说这是网络延迟。但我知道不是。那种感觉……就像你正在屏幕前做的那样。

关于“尤舍”的发现

我在书房的垃圾桶里翻到了一张被撕碎的权限说明书。

上面写着：“当 User 意识到自己是 User 时，实验即进入收割阶段。”

我不懂什么是收割。我只知道，他把我的每个房间都装了传感器。包括我现在写字的地方。
